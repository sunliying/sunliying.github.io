    依赖注入的几种方式

    为了使压缩js代码时注入的依赖不发生错误，最好单独列出所注入的依赖

  1. create a $inject property

    function personsCtrl( $scope,$http){...}
personCtrl.$inject = ['$scope','$http'];
personApp.controller('personCtrl',personCtrl);

  2. an inline annotation

    function PersonCtrl($scope,$http){}
   personApp.controller('personCtrl',['$scope','$http',personCtrl]);

  3. provide the construction function inline as an anonymous function

    personApp.controller('personCtrl',['$scope','$http',function($scope,$http){}]);
