选取对象的属性名和值

發表於 2016-02-26   |   分類於 js学习   |  

1. 不知道对象的具体属性

从一个对象中选取属性名

`for( var value in person){
                console.log( value );
            }`

从一个对象中选取属性值

`for( var value in person){
                console.log( person[value] );
            }`

2. 不知道对象的具体属性

从数组中选取对象，读取对象的属性值

`for ( var person in persons ){
        console.log( person.name )；
        console.log( person.gender );
    }`
